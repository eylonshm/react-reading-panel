# Main Features

* Change font size
* Change line spacing
* Change letter spacing
* Switch between light/dark mode
* More features to come soon!

# Usage

## Target by Id

```jsx
      <ReadingPanel targetId="sample"></ReadingPanel>
        <p id="sample">
            Text to be changes
        </p>
```


## Target by class

```jsx
      <ReadingPanel targetClass="sample"></ReadingPanel>
        <p className="sample">
            Text to be changes
        </p>
```
